# 시공OS · 세움 통합 플랫폼

세움 시공팀이 그동안 엑셀로 관리하던 **현장·공정·작업·자재·준공서류** 자료를 웹에서 편하게 보고 관리할 수 있게 만든 사내 도구입니다.

빌드 도구·인터넷 연결 없이 **`index.html`만 열면 바로 동작**하는 순수 정적 웹앱이라, 나중에 세움 통합 플랫폼 어디에든 폴더째 넣어 임베드할 수 있습니다.

---

## 실행 방법

파일을 더블클릭하거나, 폴더에서 간단한 정적 서버를 띄웁니다.

```bash
# 방법 1: 그냥 index.html 더블클릭

# 방법 2: 로컬 서버 (권장)
python3 -m http.server 8080
# → 브라우저에서 http://localhost:8080
```

> 세움 플랫폼에 넣을 때는 이 폴더(`index.html`, `assets/`)를 원하는 경로에 두고 iframe 또는 하위 페이지로 연결하면 됩니다.

## 기능

| 메뉴 | 내용 |
|------|------|
| **대시보드** | 전체/진행중/준공완료/정체주의 현장 수, 공정 단계별 분포, 이번 주 예정 일정, 정체 현장, 처리 대기 작업 |
| **현장 현황** | 현장 119개 카드형 목록 · 주소/건축주/연락처/종류 검색 · 공정·상태 필터 · 상세 모달(정보·공정 진행·일정 타임라인·준공서류 체크리스트) |
| **작업 · 체크리스트** | 인수인계·통화 리스트 + 일일 업무보고 · 완료 체크 · 검색/필터 · 항목 추가 |
| **자재 물량** | 현장별 물량/타일 산출서(자재 합계 + 산출 상세식) |
| **표준 공정표** | 시공부 표준 14단계 공사일정표(가설→준공청소) |
| **엑셀 불러오기** | 엑셀 셀을 복사해 붙여넣으면 작업 항목으로 대량 추가 |

## 데이터

원본 엑셀 6종을 파싱해 `assets/js/data.js`(`window.SEUM_DATA`)로 만들었습니다.

- 현장 관리 시트 → 현장 119개 + 일자별 일정
- 시공 공정표 → 표준 14단계 공정 + 현재 공정 자동 추정 키워드
- 일일 업무보고 → 작업 체크리스트
- 물량 산출서 / 타일 물량 산출서 → 자재 물량
- 준공 업무 체크리스트 → 준공서류 표준 목록(주택 / 체류형쉼터)

### 현재 공정 자동 추정
각 현장의 "최근 진행상황" 문구를 키워드 매칭해 14단계 중 현재 공정을 추정합니다(예: `타일 …투입` → 타일공사). 상세 모달의 드롭다운에서 **수동으로 정확히 지정**할 수 있으며, 지정값이 우선합니다.

### 편집 저장
체크 상태·공정 지정·추가 항목 등 사용자의 편집은 브라우저 **localStorage**에 저장됩니다. **원본 엑셀/데이터 파일은 변경되지 않습니다.** 사이드바 하단의 *샘플 데이터로 초기화*로 편집을 모두 되돌릴 수 있습니다.

## 구조

```
index.html            # 화면 뼈대 + 스크립트 로드
assets/
  css/styles.css      # 스타일 (외부 의존성 없음)
  js/
    data.js           # 엑셀에서 추출한 실제 데이터 (자동 생성)
    store.js          # 상태 관리 + localStorage 오버레이
    app.js            # 화면 렌더링 / 라우팅
```
